# Improvements for SES Think Tank

Make the following improvements. Do NOT modify test files. After each fix, verify the file still imports correctly. DO NOT run tests.

## 1. Fix shell execution on Windows

`plugins/tools/__init__.py` line 119: `["sh", "-c", cmd]` doesn't work on Windows. Use `["cmd", "/c", cmd]` on Windows and `["sh", "-c", cmd]` on Linux/macOS. Detect via `os.name == "nt"`.

## 2. Fix conn.close() on pooled connections

The security fix introduced `conn.close()` calls in `routers/intelligence.py` line 50. This breaks the connection pool. Remove ALL `conn.close()` calls since `db.py` pool manages lifecycle.

## 3. Add missing `from pathlib import Path` imports

Check `routers/intelligence.py` — it uses `Path` but may not import it.

## 4. Fix deprecated `on_event` startup

`app.py` line 3097: `@app.on_event("startup")` is deprecated. Migrate to lifespan context manager pattern:

```python
from contextlib import asynccontextmanager

@asynccontextmanager
async def lifespan(app: FastAPI):
    init_memory_db()
    load_sessions_from_disk()
    from routers import (core_router, plugins_router, intelligence_router, ...)
    app.include_router(core_router)
    # ... include all routers
    yield

app = FastAPI(lifespan=lifespan, ...)
```

## 5. Add rate limit cleanup

`app.py` line 61, 3130: Rate limit store (`rate_limit_store` dict) never purges old entries. Add a periodic cleanup that removes entries older than 1 hour.
